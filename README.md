# 🐔 The Chicken Lady — "What Chicken Are You?" Quiz

A fun, AI-powered personality quiz hosted on Cloudflare Pages.
Visitors answer 6 questions and get a unique chicken breed result generated by Groq AI.

---

## 📁 File Structure

```
chicken-lady-quiz/
├── index.html   ← The quiz page (this is what visitors see)
├── worker.js    ← The Cloudflare Worker proxy (keeps your API key secret)
└── README.md    ← This file
```

**Important:** Your Groq API key is NEVER stored in these files.
It lives only inside Cloudflare as an encrypted Secret.

---

## 🧰 Tech Stack

| Service | What It Does | Dashboard |
|---|---|---|
| **GitHub** | Stores your files, tracks changes | github.com |
| **Cloudflare Pages** | Hosts `index.html` as a live website | dash.cloudflare.com |
| **Cloudflare Workers** | Runs `worker.js` to secretly call Groq | dash.cloudflare.com |
| **Groq AI** | Generates the personality results | console.groq.com |

## 🚀 First-Time Setup (do this once)

### Step 1 — Upload files to GitHub
1. Go to **github.com** → click **+** → **New repository**
2. Name it `chicken-lady-quiz`, set to **Public**, click **Create**
3. Click **"uploading an existing file"**
4. Drag in `index.html`, `worker.js`, and `README.md`
5. Click **Commit changes**

---

### Step 2 — Deploy the Worker on Cloudflare (hides your API key)
1. Go to **dash.cloudflare.com**
2. Click **Workers & Pages** → **Create** → **Create Worker**
3. Name it `chicken-lady-proxy` → click **Deploy**
4. Click **Edit code** → delete everything → paste contents of `worker.js`
5. Click **Deploy**

**Add your secret Groq API key:**

6. Go back to your Worker → **Settings** → **Variables and Secrets**
7. Click **Add** → type **Secret**
8. Name: `GROQ_API_KEY` (must be exact)
9. Value: paste your Groq API key
10. Click **Deploy**
11. Copy your Worker URL — it looks like:
    `https://chicken-lady-proxy.YOURNAME.workers.dev`

---

### Step 3 — Update index.html with your Worker URL
1. Open `index.html` in any text editor (Notepad is fine)
2. Find this line near the top of the `<script>` section:
   ```
   const WORKER_URL = "https://YOUR-WORKER-NAME.YOUR-SUBDOMAIN.workers.dev";
   ```
3. Replace it with your actual Worker URL from Step 2
4. Save the file
5. Go to GitHub → click `index.html` → click the pencil ✏️ icon → paste the updated file → **Commit changes**

---

### Step 4 — Deploy the quiz page on Cloudflare Pages
1. Go to **dash.cloudflare.com** → **Workers & Pages** → **Create** → **Pages**
2. Click **Connect to Git** → authorize GitHub → select `chicken-lady-quiz`
3. Leave all build settings blank
4. Click **Save and Deploy**
5. Your live quiz URL will be: `https://chicken-lady-quiz.pages.dev` 🎉

---

### Step 5 — Embed in Google Sites
1. Open your site editor at **chickens.rlbdesigns.com**
2. Click **Insert** → **Embed** → paste your `pages.dev` URL
3. Save and publish

## ✏️ How to Make Edits

Every time you update `index.html` on GitHub, Cloudflare **automatically republishes** within ~30 seconds. No extra steps needed.

### Editing the quiz questions
1. Open `index.html` in a text editor
2. Find the `const questions = [` section
3. Each question looks like this:
   ```javascript
   {
     text: "Your question goes here?",
     answers: [
       { emoji: "🌅", text: "Answer option one" },
       { emoji: "🌿", text: "Answer option two" },
       { emoji: "🛌", text: "Answer option three" },
       { emoji: "👑", text: "Answer option four" },
     ]
   },
   ```
4. Edit the text, add/remove questions (keep the commas!)
5. Save → upload to GitHub → Cloudflare updates automatically

### Editing the AI personality prompt
1. Open `worker.js` in a text editor
2. Find the section that says `const prompt =` (inside `index.html`, in the `getResult()` function)
3. Edit the instructions to change how the Chicken Lady speaks or what she focuses on
4. Save → upload to GitHub

### Changing colors or fonts
1. Open `index.html`
2. Find the `:root {` section near the top — this controls all the colors:
   ```css
   --yolk:    #F5A623;   /* orange/gold */
   --comb:    #E8433A;   /* red */
   --feather: #6DBE6C;   /* green */
   --sky:     #FFF8ED;   /* page background */
   --earth:   #5C3D1E;   /* dark brown text */
   ```
3. Change any hex color code to your preferred color
4. Save → upload to GitHub

## 🔑 Managing Your Groq API Key

Your key is stored **only** in Cloudflare Secrets — never in any file.

### If you need to replace your key:
1. Go to **console.groq.com** → **API Keys** → create a new key
2. Go to **dash.cloudflare.com** → **Workers & Pages** → `chicken-lady-proxy`
3. Click **Settings** → **Variables and Secrets**
4. Find `GROQ_API_KEY` → click **Edit** → paste the new key → **Deploy**
5. Delete the old key from console.groq.com

### If GitHub flags your key:
This means the key was accidentally put inside a file.
1. Immediately go to **console.groq.com** and delete the exposed key
2. Create a new key
3. Store the new key only in Cloudflare Secrets (never paste it into a file)

## 🐛 Troubleshooting

| Problem | Fix |
|---|---|
| Quiz loads but AI result never shows | Check your Worker URL in `index.html` is correct |
| "Worker error" message | Check your `GROQ_API_KEY` secret is saved in Cloudflare |
| Changes not showing after GitHub update | Wait 60 seconds and hard-refresh (Ctrl+Shift+R) |
| Google Sites embed shows blank | Make sure your Cloudflare Pages URL is using `https://` |
| Groq key flagged by GitHub | Delete the key, create a new one, store it only in Cloudflare Secrets |
